# Background Music

`background.mp3` is a tiny placeholder tone so the music player works out of
the box. Replace it with your own royalty-free instrumental track (same
filename, `background.mp3`), or update the `<source>` path in
`src/components/MusicPlayer.jsx` if you'd like to use a different filename.

Recommended:
- Instrumental (no vocals), 2–4 minute loop
- MP3, 128–192kbps, under ~5MB for fast loading
- Make sure you have the rights/license to use the track on a public site

This `README.md` file is safe to delete once you've replaced the track.
