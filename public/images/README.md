# Image Placeholders

Every image below is an auto-generated, on-theme placeholder (gold / ivory /
forest-green) so the site looks complete out of the box. Replace each file
with your own photo **using the exact same filename** and everything will
update automatically — no code changes required.

| File | Used for | Recommended size |
|---|---|---|
| `hero-bg.jpg` | Full-screen hero background | 1920×1080 (landscape) |
| `groom.jpg` | Groom portrait in "Our Story" | 900×1200 (portrait, 3:4) |
| `bride.jpg` | Bride portrait in "Our Story" | 900×1200 (portrait, 3:4) |
| `gallery1.jpg` | Gallery grid, image 1 | 1200×1500 (portrait, 4:5) |
| `gallery2.jpg` | Gallery grid, image 2 | 1200×1500 (portrait, 4:5) |
| `gallery3.jpg` | Gallery grid, image 3 | 1200×1500 (portrait, 4:5) |
| `gallery4.jpg` | Gallery grid, image 4 | 1200×1500 (portrait, 4:5) |

Tips:
- Keep files under ~500KB each for fast loading (export as JPEG, quality ~80).
- Portrait photos work best for `bride.jpg` / `groom.jpg` since they render
  inside an arched frame.
- This `README.md` file is safe to delete once you've replaced the images.
